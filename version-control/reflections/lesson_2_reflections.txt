How is the staging area different from the working directory and the repository? What value do you think it offers?

  The staging area is in an initialized directory and contains the files to be committed.  It allows you to make a commit while stile having changes saved in your working directory that aren't ready to be committed yet.

  check the staging area vs the working directory ( git diff )

  Branches allow multiple people to work at once and commit their changes.  Also another method of organizing versions of the game
